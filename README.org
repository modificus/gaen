* Building clean on OSX Maverick

Install gcc 4.8 through homebrew

Add a setcc.sh in root directory containing something like:
export CC=/usr/local/bin/gcc-4.8
export CXX=/usr/local/bin/g++-4.8

Run bootstrap.sh

Make in build/Debug
